== Installation ==

1.    Copy Specification BenignSource in your plugin directory
2.    In your WordPress administration, go to the Plugins page
3.    Activate the Specification BenignSource plugin and a subpage for the plugin will appear in your Manage menu.
4.    Go to the Products and make the Full Specification Product, Delivery & Returns Policy
5.    Enjoy!
